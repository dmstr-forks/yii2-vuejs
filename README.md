# yii2-vuejs


This is Vue.js library for Yii2 framework 

## Install 
Via Composer
``` bash 
$ composer require "dmstr-forks/prawee-yii2-vuejs"
```

## Usage
``` php
use prawee\vuejs\VueJsAsset;
```
On your view 
``` php
VueJsAsset::register($this);
```
On your controller
``` php
VueJsAsset::register($this->view);
```
